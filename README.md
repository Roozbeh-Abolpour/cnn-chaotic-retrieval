\# Controlled Chaotic Neural Network for Pattern Retrieval



\## Overview

This project implements a controlled chaotic neural network (CNN) for associative memory and pattern retrieval.



The model combines nonlinear neural dynamics with a control mechanism that regulates refractoriness states, enabling robust retrieval from noisy inputs.



\## Features

\- Storage of multiple binary image patterns

\- Controlled chaotic dynamics

\- Noise-robust retrieval

\- Time-evolution visualization



\## Example



Run:



&#x20;   cd examples





\## Results



The network successfully retrieves structured patterns from noisy initial conditions. The dynamics exhibit:



\- progressive pattern organization

\- transient switching behavior

\- bounded controlled evolution
